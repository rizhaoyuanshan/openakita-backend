## 二、核心传播卖点（按优先级排列）

### 卖点 1：多个 AI 组团干活——多 Agent 协作

**痛点**：单个 AI 能力有限，复杂任务做到一半就迷失了，你只能一件件交代、反复催
**解法**：OpenAkita 内置多 Agent 协作系统——一个"总指挥"自动把任务拆分给多个专业 Agent，它们并行工作、互相接力，你只需要说一句话

**核心能力**：
- **专业分工**：技术问题交给"技术专家"Agent，写文案交给"文案"Agent，各司其职
- **并行处理**：多个 Agent 同时执行不同子任务，效率翻倍
- **自动接力**：一个 Agent 搞不定，自动移交给更合适的
- **可视化追踪**：神经网络仪表盘实时展示每个 Agent 的工作状态
- **深度防护**：5 层委派深度控制，防止任务递归失控

**话术示例**：
- "一句话下去，3 个 AI 同时开工——一个写代码、一个写文档、一个测试，我去喝杯咖啡"
- "别人用一个 AI，我用一个 AI 团队"
- "总指挥自动分活、自动盯进度、自动汇总结果——真正的 AI 团队协作"
- "钉钉里说一句'帮我做个竞品分析'，后面是 3 个 Agent 在同时搜索、整理、写报告"

**配合计划模式更强**：
- 大项目自动拆成小步骤，每个步骤可以分给不同 Agent
- 实时进度条，随时看到走到了哪一步
- 某一步卡住了？自动回退换方案，不用你操心

### 卖点 2：真的能"做事"，不是只会聊天

**痛点**：市面上的 AI 助手只能一问一答，复杂任务搞不定
**解法**：OpenAkita 能上网搜索、操作电脑、读写文件、浏览器自动化、定时提醒……89 种内置技能

**话术示例**：

- "让 AI 真正动手干活，不是只动嘴。"

### 卖点 3：零门槛上手——全程不用敲一行命令

**痛点**：同类开源 AI 助手普遍配置繁琐，要改配置文件、跑命令行、装各种依赖，劝退了 90% 的普通用户
**解法**：OpenAkita 全流程图形化配置——下载安装包 → 打开引导向导 → 填 API Key → 开始用。从安装到对话，**全程零命令行、零代码**

**核心体验**：
- **一键安装**：Windows / macOS / Linux 原生安装包，双击即装
- **引导向导**：首次启动自动弹出配置向导，一步步引导你配好所有选项
- **可视化配置**：模型、IM 通道、技能、MCP、定时任务……全部在桌面界面点击配置
- **环境自理**：Python 运行时自动隔离，依赖自动安装，不污染你的系统环境
- **中国用户友好**：镜像源自动切换，不怕网络问题

**话术示例**：
- "别的 AI 助手：先装 Python，再 git clone，然后 pip install，接着改 yaml……OpenAkita：双击安装，打开就用。"
- "你妈都能配——不是开玩笑，真的全程不用敲命令行"
- "从下载到跟 AI 对话，5 分钟"
- "别的开源 AI 助手要你改配置文件，OpenAkita 给你一个向导界面"

## 八、竞品差异化话术

| vs 竞品 | OpenAkita 优势 |
|---------|---------------|
| vs ChatGPT / Kimi / 豆包 | 多 Agent 协作不是单打独斗；不止聊天，能操作电脑、管文件、定时任务；支持多模型不锁厂商；数据本地 |
| vs 其他开源 AI 助手 | **全图形化配置，零命令行上手**；其他方案需要改 yaml + 跑命令行 + 手动装依赖，OpenAkita 双击安装 → 向导引导 → 5 分钟开聊；环境自动隔离不污染系统 |
| vs Cursor / Windsurf | 不限于代码场景，覆盖日常办公全场景；多 Agent 分工协作；接入 6 大 IM 通道 |
| vs AutoGPT / MetaGPT | **不需要命令行**，有精美桌面 UI + 引导向导；多 IM 平台接入；Agent 实例池化 + 故障切换更稳定；记忆系统更成熟 |
| vs Coze / Dify | 不是 workflow 拖拽，是真正自主推理 + 自动分工的多 Agent 系统；开源可控；不收平台费 |
| vs 企业级 AI SaaS | 开源免费、数据本地、用自己的 Key；多 Agent 协作内置，无月费无人头费；部署简单不需要运维 |
